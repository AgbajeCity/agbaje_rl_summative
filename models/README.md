# Saved Models Directory

This directory contains trained RL agent model files saved during hyperparameter tuning.

## Structure

```
models/
├── dqn/                    # DQN model checkpoints
│   ├── DQN_baseline.zip
│   ├── DQN_small_batch.zip  # Best DQN model (reward: 348.12)
│   └── ...                 # All 10 DQN configurations
└── pg/                     # Policy Gradient model checkpoints
    ├── PPO_baseline.zip
    ├── PPO_low_lr.zip       # Best PPO model (reward: 167.30)
    ├── RF_optimal.zip       # Best REINFORCE model (reward: 163.30)
    ├── A2C_high_ent.zip     # Best A2C model (reward: 161.57)
    └── ...                 # All 30 PG configurations
```

## Best Models Performance

| Model | Algorithm | Mean Reward | Std |
|-------|-----------|-------------|-----|
| DQN_small_batch.zip | DQN | 348.12 | 145.88 |
| PPO_low_lr.zip | PPO | 167.30 | 41.64 |
| RF_optimal.zip | REINFORCE | 163.30 | 38.30 |
| A2C_high_ent.zip | A2C | 161.57 | 26.33 |

## Loading Models

```python
from stable_baselines3 import DQN, PPO, A2C
# Load best DQN model
model = DQN.load("models/dqn/DQN_small_batch")
# Run simulation
python main.py  # Uses best model automatically
```

Models are saved automatically during training and can be regenerated by running the training scripts.
